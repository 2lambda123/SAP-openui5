Manual Integration Test: Multiple Inline Creation Rows Grid

DESCRIPTION

Test of the Multiple Inline Creation Rows Grid application.

PREPARATION

To check console errors open the developer tools before starting the application and check with each step, that there are no errors and warnings.

Testing with mock data

Start the Multiple Inline Creation Rows Grid application with following URL (adjust host and port if necessary): http://<host>:<port>/test-resources/sap/ui/core/demokit/sample/common/index.html?component=odata.v4.MultipleInlineCreationRowsGrid

REALIZE and CHECK

  1. Check: A list with three products appears.
  2. Select the first product with ID 10.
  3. Check: A table with title "3 Parts" appears. It contains three product parts, followed by three empty rows.
  4. Check: The delete button is active for the parts and inactive for the empty rows.
  5. Enter "99" into the ID field of the first empty row and press Enter.
  6. Check: The delete button of the row becomes active.
  7. Check: The table title changes to "4 Parts".
  8. Check: A new empty row appears.
  9. Enter "100" into the ID field of the first empty row and press Enter.
  10. Check: An error message "Key exists already" appears.
  11. Check: The delete button of the row becomes active.
  12. Check: The table title changes to "5 Parts".
  13. Check: A new empty row appears.
  14. Enter "101" into the ID field of the first empty row and press Enter.
  15. Check: The delete button of the row becomes active.
  16. Check: The table title changes to "6 Parts".
  17. Check: A new empty row appears.
  18. Click on the delete button of the fifth row with the ID "100"
  19. Confirm the dialog with clicking on "Ok"
  20. Check: The table title changes to "5 Parts".
  21. Check: The row disappears.
